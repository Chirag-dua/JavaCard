DWORD __stdcall SDSCListRemovableDrives ( OUT char *  pszRemovableDrives,
IN OUT DWORD *  pulRemovableDrivesLen,
OUT DWORD *  pulRemovableDriveNum 
)

Get the list of all the removable drives. This function won't send GO-Trust commands to all the removable devices.

Parameters
pszRemovableDrivesReceives a multi-string that lists all the removable drives.
If this value is NULL, SDSCListRemovableDrives ignores the buffer length supplied in pulRemovableDrivesLen, writes the length of the buffer that would have been returned if this parameter had not been NULL to pulRemovableDrivesLen, and returns a success code.
pulRemovableDrivesLenSupplies the length of the pszRemovableDrives buffer in characters, and receives the actual length of the multi-string structure, including the trailing null character.
pulRemovableDriveNumReceives the number of all the removable drives.
Returns
Returns 0 on success; otherwise, returns an error code.

DWORD __stdcall SDSCGetSCIOType ( IN HANDLE  hDevice,
OUT DWORD *  pulSCIOType 
)

SDSCGetSCIOType get the communication protocol type of the smart card.

Parameters
hDeviceSupplies the device handle obtained by calling SDSCConnectDev().
pulSCIOTypeReceives the communication protocol type of the smart card.
It can be one of the following:
  • SDSC_SCIO_7816
  • SDSC_SCIO_SPI_V1
  • SDSC_SCIO_SPI_V2
  • SDSC_SCIO_SPI_V3
Returns
Returns 0 on success; otherwise, returns an error code.
