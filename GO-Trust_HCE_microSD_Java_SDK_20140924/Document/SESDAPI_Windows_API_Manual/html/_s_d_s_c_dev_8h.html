DWORD __stdcall SDSCPPS ( IN HANDLE  hDevice,
IN BYTE  bRate,
OUT BYTE *  pbPPS,
IN OUT DWORD *  pulPPSLen 
)

PPS exchange.

Parameters
[in]hDeviceSupplies the device handle obtained by calling SDSCConnectDev().
[in]bRateSupplies the baud rate(Fi/Di) in the PPS request.
[out]pbPPSReceives the actual Fi/Di applied by the device.
[in,out]pulPPSLenSupplies the length of the pbPPS buffer in bytes, and receives the length of returned data.
Returns
Returns 0 on success; otherwise, returns an error code.
