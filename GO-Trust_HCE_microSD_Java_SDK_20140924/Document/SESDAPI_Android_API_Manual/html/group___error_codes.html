final int com.gotrust.sesdapi.SESDAPI.SDSC_DEV_NO_FILE_ERROR = 0x0F00002C
static

The specified file in the device has not been created.

If the error occurs in SDSCListDevs() or SDSCConnectDev(), users have to call SDSCCreateIoFile() to create the I/O file.

final int com.gotrust.sesdapi.SESDAPI.SDSC_DEV_COMMUNICATION_ERROR = 0x0FF00000
static

The base error code for internal communication errors occur between the application and the device.


More specifically, an error code is composed of (SDSC_DEV_COMMUNICATION_BASE_ERROR | two-byte abnormal communication state code).

final int com.gotrust.sesdapi.SESDAPI.SDSC_SYSTEM_CALL_ERROR_BASE = 0x0FE00000
static

The base error code for errors from system calls.

Whenever a function fails in a system call, it will return (SDSC_SYSTEM_ERROR_BASE | errno).
