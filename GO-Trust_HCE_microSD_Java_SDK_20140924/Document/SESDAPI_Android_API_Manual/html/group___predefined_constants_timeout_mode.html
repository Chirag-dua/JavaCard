final int com.gotrust.sesdapi.SESDAPI.SDSC_DEV_CUSTOM_FLAG_TIME_OUT = 0x80000000
static

Use the custom timeout value of the device.

The time unit is millisecond. SDSC_DEV_CUSTOM_FLAG_TIME_OUT is a flag, it must be bitwise-or'd with the custom timeout value.
Timeout value range is 0x00000000 ~ 0x7FFFFFFF.
Example: If the expected timeout is 300 ms, the parameter should be (SDSC_DEV_CUSTOM_FLAG_TIME_OUT | 300).
