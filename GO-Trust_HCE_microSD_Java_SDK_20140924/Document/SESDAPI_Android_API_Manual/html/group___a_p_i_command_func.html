final int com.gotrust.sesdapi.SESDAPI.SDSCGetDeviceInfo ( int  fd,
byte  bDeviceInfo[] 
)

Get the secure microSD device info.

The content of the returned device info depends on product type. For the device of the type SDSC_PRODUCT_TYPE_NS, the device info is its flash ID. For the device of the type SDSC_PRODUCT_TYPE_CUP, the device info is the response of the SCIF_INFO command.

Parameters
fdSupplies the device handle obtained by calling SDSCConnectDev().
bDeviceInfoReceives the device info. The buffer must not be smaller than SDSC_DEVICE_INFO_LEN.
Returns
Receives the actual length of the received device info.
Exceptions
SDSCExceptionwill be thrown if any error occurs.

final int com.gotrust.sesdapi.SESDAPI.SDSCTransmitEx ( int  fd,
byte  bCommand[],
int  off,
int  intCommandLen,
int  intTimeOutMode,
byte  bOutData[] 
)

Send a COS command to the device, and expect to receive data back from the device.

Parameters
fdSupplies the device handle obtained by calling SDSCConnectDev().
bCommandSupplies the byte array including the COS command to be written to the device.
offSupplies the offset of the COS command in command parameter.
intCommandLenSupplies the lengths of the COS command in bytes.
intTimeOutModeSpecifies the timeout to wait for a response. The timeout can set to one of the following values:
  • SDSC_DEV_DEFAULT_TIME_OUT
  • SDSC_DEV_MAX_TIME_OUT
  • SDSC_DEV_CUSTOM_FLAG_TIME_OUT: Using the custom timeout value of the device, the time unit is millisecond.
bOutDataReceives the data from the device (including two bytes COS state code). The buffer must be big enough.
Returns
Returns the actual length of received data.
Exceptions
SDSCExceptionwill be thrown if any error occurs.
