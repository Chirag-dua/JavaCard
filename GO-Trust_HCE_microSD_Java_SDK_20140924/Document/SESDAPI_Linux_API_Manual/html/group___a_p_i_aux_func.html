unsigned long SDSCGetSDKVersion ( char *  pszVersion,
unsigned long *  pulVersionLen 
)

Get the SDK version.

Parameters
pszVersionReceives the SDK version. The buffer size must not be smaller than SDSC_MAX_VERSION_LEN.
pulVersionLenSupplies the length of the pszVersion buffer in bytes, and receives the actual length of the version string, including the trailing null character.
Returns
Returns 0 on success; otherwise, returns an error code.

unsigned long SDSCGetSCIOType ( int  hDevice,
unsigned long *  pulSCIOType 
)

Get the communication protocol type of the smart card.

Parameters
hDeviceSupplies the device handle obtained by calling SDSCConnectDev().
pulSCIOTypeReceives the communication protocol type of the smart card. It will be one of the following values:
  • SDSC_SCIO_7816
  • SDSC_SCIO_SPI_V1
  • SDSC_SCIO_SPI_V2
  • SDSC_SCIO_SPI_V3
Returns
Returns 0 on success; otherwise, returns an error code.
