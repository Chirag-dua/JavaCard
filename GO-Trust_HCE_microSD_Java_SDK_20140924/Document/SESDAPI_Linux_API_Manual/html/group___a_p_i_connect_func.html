unsigned long SDSCListDevs ( char *  pszDrives,
unsigned long *  pulDrivesLen,
unsigned long *  pulDriveNum,
char *  pszIoFileName 
)

Get the list of all the devices according to SDSC_DEVICE_PATH_NAME. If SDSC_DEVICE_PATH_NAME is not assigned, it will search available devices by checking all mounted devices.

Parameters
pszDrivesReceives a multi-string that lists all the devices.
If this value is NULL, SDSCListDevs ignores the buffer length supplied in pulDrivesLen, writes the length of the buffer that would have been returned if this parameter had not been NULL to pulDrivesLen, and returns a success code.
pulDrivesLenSupplies the length of the pszDrives buffer in bytes, and receives the actual length of the multi-string structure.
pulDriveNumReceives the number of devices.
pszIoFileNameSupplies the I/O file name. The communication channel between host applications and the secure microSD will be established based on the specified I/O file on the microSD.
Returns
Returns 0 on success; otherwise, returns an error code.
See also
SDSCErr.h
unsigned long SDSCConnectDev ( char *  pszDrive,
int *  phDevice,
char *  pszIoFileName 
)

Establishe a connection between the calling application and the device.
If the smart card has not been reset, this function will do SDSCResetCard() automatically.

Parameters
pszDrivePointer to a null-terminated string that specifies the device drive name.
phDeviceReceives device file descriptor that identifies the connection to the device.
pszIoFileNameSupplies the I/O file name. The communication channel between host applications and the secure microSD is established by using a special I/O file under the root directory of the microSD.
Returns
Returns 0 on success; otherwise, returns an error code.
See also
SDSCErr.h
